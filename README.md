# spring-core-rce
Spring-Core-Rce-Tools Go

##  工具介绍

1. 内置写入的路径是 webapps/ROOT/,可以使用参数 -d 来指定上传的路径
2. 攻击成功之后，会写入一个回显的 jsp Webshell
3. 每次上传的webshell名称都是随机
